# Emotion Detection from Video Using Machine Learning

## Quick start
### Backend
```bash
cd backend
python3.11 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Frontend
```bash
cd frontend
npm install
npm run dev
```
